# AI混剪素材生成功能PRD20260330

## 1. 需求背景

黛莱皙（DINESSR）深度绑定短视频与直播电商赛道，创始人以"猫七七阿姨"IP身份在快手、抖音等平台持续运营，随着业务规模持续扩大、产品线不断扩充（2024年发布木兰系列2.0、玉翡系列），黛莱皙面临内容生产压力的急剧上升。


| 痛点      | 具体表现                             |
| ------- | -------------------------------- |
| 内容需求量大  | 多平台（快手/抖音/视频号）并行运营，每日需产出大量差异化短视频 |
| 素材生产成本高 | 产品拍摄、模特出镜、场景搭建均需专业团队，成本难以线性扩展    |
| 品牌调性维护难 | "科技新国风"东方美学风格需在海量内容中保持一致性        |
| 新品发布响应慢 | 从产品拍摄到上线内容链路长，难以快速响应新品发布节奏       |
| 工具链条割裂  | 视频剪辑、AI图片、数字人、配音等需在多个工具间切换，效率损耗大 |


**核心矛盾：** 品牌已到达规模化运营阶段，但内容生产能力仍依赖重人力的传统模式，难以支撑7000万粉丝体量所需的高频优质内容输出。

## 2. 建设目标

本次素材生成模块的建设，旨在为黛莱皙构建一套**专属的AI素材生产中台**，将图片生成、视频生成、数字人、配音生成能力统一整合至现有AI混剪工具中，实现：

- **降本**：减少对外部拍摄团队、配音演员的依赖
- **提效**：新品上线内容生产周期从天级压缩至小时级
- **保质**：内置品牌调性参数，确保"科技新国风"风格的内容一致性
- **扩量**：支持批量化素材生产，满足多平台多场景并行内容需求

## 3. 目标用户


| 用户角色   | 核心诉求                     | 主要使用模式       |
| ------ | ------------------------ | ------------ |
| 短视频运营  | 快速产出符合品牌调性的日常内容          | Agent模式、图片生成 |
| 电商直播团队 | 批量制作新品推广视频与直播素材          | 视频生成、数字人     |
| 内容策划   | 快速验证创意方案，降低试错成本          | Agent模式      |
| 品牌营销   | 重要节点（新品发布、代言人活动）的高质量内容生产 | 视频生成、配音生成    |


## 4. 预期收益

- **内容产能提升3倍以上**，支撑多平台并行高频发布
- **素材生产成本降低40%**，减少对外部拍摄和配音资源的依赖
- **新品内容响应周期**从5-7天压缩至1天内
- 为黛莱皙持续扩大规模（年GMV向更高目标迈进）提供内容基础设施支撑

## 5. 需求概况

### 5.1 功能模块总览

素材生成模块共包含 **5 个功能模式**，集成于AI混剪工具现有工作台内，通过底部工具栏快速切换：


| 模式       | 功能定位                       | 典型使用场景             |
| -------- | -------------------------- | ------------------ |
| Agent 模式 | 输入创意描述，AI自动编排分镜、批量生成全套素材   | 新品上市快速出片、系列内容批量生产  |
| 图片生成     | 根据文本或参考图生成静态画面素材           | 产品场景图、封面图、电商主图     |
| 视频生成     | 生成动态视频片段，支持首尾帧控制和运镜设定      | 产品展示片段、氛围视频、转场素材   |
| 数字人      | 生成虚拟主播讲解视频，支持音色选择与动作控制     | 产品讲解、品牌故事、活动推广     |
| 配音生成     | 将文本转为高质量语音，支持猫七七阿姨多音色与情感调节 | 视频旁白、广告配音、多平台差异化内容 |


### 5.2 核心用户流程

```
输入创意/需求
    ↓
选择生成模式（Agent / 图片 / 视频 / 数字人 / 配音）
    ↓
配置生成参数（模型、比例、时长、风格等）
    ↓
AI 生成素材（支持多方案并行生成）
    ↓
预览 & 筛选结果
    ↓
一键添加至时间轴 / 导出素材库
```

### 5.3 功能边界说明

**本期包含：**

- 5种素材生成模式的完整交互与生成能力
- 统一素材库：生成结果自动归档，支持按类型/项目筛选
- 生成历史记录与参数复用
- 素材与时间轴的一键联动（生成后直接插入剪辑轨道）

**本期不包含：**

- 生成素材的二次编辑（如AI修图、视频补帧）——规划至下一期

## 6. 需求详情

### 6.1 Agent 模式（智能创作助手）

#### 6.1.1 功能定位

通过自然语言描述创意需求，AI自动理解并规划工作，规划确认后自动按照规划完成任务。

#### 6.1.2 核心功能

##### （1）创意输入

- 多行文本框，支持 500 字以内的创意描述
- 支持上传参考主体（图片、视频），或通过 @ 方式引用已有素材
- 支持自定义生成偏好（图片/视频生成的模型选择等）
- 智能提示：引导用户输入关键信息，提升 AI 理解准确率

##### （2）AI 规划与确认

- AI 自动解析输入内容，生成工作规划（分场景、分素材类型）
- 用户可预览并确认规划方案，支持修改后再执行
- 确认后自动调度各模块（图片/视频/数字人/配音）批量执行

> Agent 模式的详细生成逻辑与各子模块保持一致，参照 6.2-6.5 节。

### 6.2 图片生成

#### 6.2.1 功能定位

快速生成符合品牌调性的静态画面素材，适用于产品场景图、封面图、电商主图等场景。

#### 6.2.2 核心功能

##### （1）输入方式

- **文本描述**：200字以内，支持中英文；智能补全常用描述词
  - 示例："木兰系列精华液产品特写，东方瓷器质感背景，柔和光线"
- **参考图上传**：支持拖拽上传，最多 4 张；用途包括风格参考、构图参考、产品参考；可调整参考强度（10%-90%）
- **引用主体**：支持引用数字人作为参考主体

##### （2）参数配置

| 参数 | 选项 | 说明 |
|------|------|------|
| 模型选择 | 图片5.0 Lite / Pro / Ultra | Lite快速，Ultra精细 |
| 画面比例 | 1:1 / 16:9 / 9:16 / 4:3 / 3:4 | 根据使用场景选择 |
| 分辨率 | 标清 / 高清2K / 超清4K | 影响生成耗时 |
| 生成数量 | 1-4张 | 一次生成多个方案 |
| 创意程度 | 滑块：保守 ← → 创新 | 控制AI发挥空间 |
| 细节程度 | 滑块：简约 ← → 精细 | 控制画面细节丰富度 |
| 色彩风格 | 暖色调/冷色调/自然/黑白 | 整体色调控制 |

##### （3）生成结果展示

- 网格布局展示生成的图片（2x2 或 1x4）
- 每张图片悬浮显示操作按钮：下载到本地、保存到素材库（选择文件夹）、重新生成
- 支持选中多张批量操作

##### （4）特色功能

- **智能扩图**：选中图片后可向上/下/左/右扩展画面
- **局部重绘**：圈选区域重新生成（如替换背景、调整产品角度）
- **风格迁移**：上传风格参考图，应用到生成结果
- **品牌水印**：自动添加黛莱皙品牌水印（可选）

#### 6.2.3 使用场景示例

| 场景 | 输入描述示例 | 参数配置 |
|------|-------------|---------|
| 产品主图 | "木兰多肽精华液瓶身特写，白色大理石台面，柔和侧光" | 1:1，Ultra，写实风格 |
| 封面图 | "东方女性使用护肤品，温馨卧室场景，早晨阳光" | 16:9，Pro，科技新国风 |
| 氛围图 | "中国传统瓷器纹理背景，金色线条点缀，高级感" | 9:16，Lite，插画风格 |

### 6.3 视频生成

#### 6.3.1 功能定位

生成动态视频片段，支持首尾帧控制和运镜设定，适用于产品展示、氛围视频、转场素材等场景。

#### 6.3.2 核心功能

##### （1）输入方式

- **文本描述**：描述视频内容和运镜方式
  - 示例："精华液瓶身旋转展示，镜头缓慢推进，背景虚化"
- **首尾帧设置**：
  - 仅首帧：上传起始画面，AI生成后续动态
  - 仅尾帧：AI生成过程，最终到达指定画面
  - 首尾帧同时控制：精确控制起止状态
  - 图片来源：本地上传 / 从图片生成结果选择
- **全能参考**：上传最多12个参考素材，支持自由组合图、文、音、视频多元素；可通过 @ 方式引用，例如"@图片1 模仿 @视频1 的动作，音色参考 @音频1"
- **参考主体**：支持引用数字人作为参考主体

##### （2）参数配置

| 参数 | 选项 | 说明 |
|------|------|------|
| 模型选择 | Seedance 2.0 Fast / Standard / Pro | Fast=4秒，Standard=8秒，Pro=16秒 |
| 画面比例 | 16:9 / 9:16 / 1:1 | 根据平台选择 |
| 分辨率 | 高清2K / 超清4K | 影响生成耗时 |
| 时长 | 5-15秒，下拉选择 | 根据模型限制 |
| 运镜类型 | 推进/拉远/平移/旋转/静止/自动 | 自动=AI智能选择 |
| 运动幅度 | 低/中/高 | 控制画面动态强度 |
| 帧率 | 24fps / 30fps / 60fps | 影响流畅度 |

##### （3）生成与预览

- 实时预览窗口：显示生成进度和预览画面
- 生成队列管理：显示当前排队位置、预估剩余时间、支持取消
- 生成完成后操作：播放预览（循环）、下载保存、添加到时间轴、重新生成

##### （4）特色功能

- **图片转视频**：上传静态图片生成动态效果（如产品旋转、背景流动）
- **视频延长**：基于现有视频生成后续片段（保持风格连贯）
- **循环视频**：自动生成首尾衔接的循环素材（适合背景视频）
- **批量生成**：同一描述生成多个版本供选择

#### 6.3.3 使用场景示例

| 场景 | 输入描述 | 参数配置 |
|------|---------|---------|
| 产品展示 | "精华液瓶身360度旋转，镜头推进特写" | 8s，Standard，推进，中运动 |
| 氛围视频 | "东方园林场景，花瓣飘落，镜头缓慢平移" | 16s，Pro，平移，低运动 |
| 转场素材 | "金色粒子从左至右流动" | 4s，Fast，平移，高运动 |

### 6.4 数字人

#### 6.4.1 功能定位

生成虚拟主播讲解视频，支持音色选择与动作控制，适用于产品讲解、品牌故事、活动推广等场景。

#### 6.4.2 核心功能

##### （1）数字人形象选择

- **预设形象库**：按类型分类（商务/时尚/科技/亲和），每个形象展示静态头像、3秒动态示例、适用场景标签
- **黛莱皙专属形象**：
  - 猫七七阿姨数字人（品牌IP）
  - 叔叔数字人
  - 猫七七数字人
  - 老四数字人
- **自定义形象**（下期功能）：上传真人照片+音频生成数字人，支持调整年龄、性别、服装风格

##### （2）内容输入

- **文本输入**：1000 字以内，支持智能断句、字数统计与预估时长、模板库（产品讲解/品牌故事）
- **音频上传**：上传音频文件，数字人口型自动同步

##### （3）其他参数

| 参数 | 选项 | 默认值 |
|------|------|-------|
| 画面比例 | 16:9 / 9:16 / 1:1 | 16:9 |
| 分辨率 | 高清2K / 超清4K | 高清2K |
| 时长 | 根据文本/音频自动计算 | 自动 |

#### 6.4.3 使用场景示例

| 场景 | 配置 |
|------|------|
| 产品讲解 | 猫七七阿姨形象 + 亲和音色 + 中景 + 产品展示台背景 |
| 品牌故事 | 创始人数字人 + 温柔音色 + 特写 + 办公室背景 |
| 活动推广 | 代言人数字人 + 热情音色 + 全景 + 演播厅背景 |

### 6.5 配音生成

#### 6.5.1 功能定位

将文本转为高质量语音，支持多音色与情感调节，适用于视频旁白、广告配音、多平台差异化内容等场景。

#### 6.5.2 核心功能

##### （1）文本输入

- 多行文本框，支持 2000 字以内
- 智能断句：自动识别停顿位置，支持手动调整

##### （2）音色选择

- **音色库**：按性别（男声/女声/童声）、按风格（磁性/甜美/浑厚/清脆/知性）、按场景（广告/纪录片/有声书/新闻/直播）多维度筛选
- **黛莱皙专属音色**：猫七七阿姨、叔叔尹海波、猫七七、尹若涵
- **试听功能**：点击音色卡片试听 5 秒示例
- **自定义音色**（下期功能）：上传音频样本克隆音色

##### （3）参数调节

| 参数 | 范围 | 说明 |
|------|------|------|
| 语速 | 0.5x - 2.0x | 滑块调节，默认 1.0x |
| 音调 | -12 ~ +12 | 调整高低音，默认 0 |
| 情感 | 平静/喜悦/悲伤/愤怒/惊讶 | 影响语气表现 |
| 停顿时长 | 短（0.3s）/ 中（0.6s）/ 长（1.0s） | 句间停顿 |
| 音量 | 0 - 100 | 整体音量控制 |

##### （4）生成与编辑

- **分段生成**：长文本自动分段（每段 500 字），支持单段重新生成和参数调整

##### （5）导出选项

- 音频格式：MP3 / WAV / M4A
- 音质：标准（128kbps）/ 高品质（320kbps）/ 无损（WAV）
- 导出方式：下载到本地 / 添加到素材库 / 直接添加到时间轴音频轨道

#### 6.5.3 使用场景示例

| 场景 | 配置 |
|------|------|
| 产品推广视频旁白 | 猫七七阿姨音色 + 热情情感 + 轻快背景音乐 |
| 品牌故事纪录片 | 知性女声 + 平静情感 + 舒缓背景音乐 |
| 直播预告配音 | 甜美女声 + 惊讶情感 + 激昂背景音乐 |
| 多平台差异化内容 | 同一文案生成 3 种音色版本（快手/抖音/视频号） |

## 7. 跨模式协同设计

### 7.1 素材流转

不同生成模式之间支持素材流转，提升工作效率：

| 流转路径 | 使用场景 |
|---------|---------|
| 图片生成 → 视频生成 | 生成的产品图直接作为视频首尾帧，添加动态效果 |
| 图片生成 → 数字人背景 | 生成的场景图作为数字人背景 |
| 视频生成 → 配音生成 | 视频时长自动传递给配音模块，确保时长匹配 |
| 配音生成 → 数字人 | 生成的配音直接用于数字人口型同步 |
| Agent模式 → 各子模式 | 分镜方案中的单个场景可跳转到对应模式精细调整 |


### 7.2 历史记录与参数复用

- 记录所有生成任务的参数配置，支持一键复用历史参数
- 支持将常用参数保存为预设（如"产品主图标准配置"、"快手短视频配置"）

- 生成的人物形象不得涉及未授权的真实人物
- 平台保留对生成内容的审查权利，违规内容将被下线处理

|------|------|
